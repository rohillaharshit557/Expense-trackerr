# Expense-trackerr
A simple command-line Expense Tracker built using Python. It allows users to add, view, and calculate daily expenses, storing data in a CSV file. Great for beginners to learn file handling, user input, and basic Python concepts.
💸 Expense Tracker (Python)

📌 Overview

This is a simple Expense Tracker built using Python. It allows users to record daily expenses, view all records, and calculate total spending using a CSV file for storage.

🛠️ Features

- Add daily expenses
- View all expense records
- Calculate total expenses
- Data stored in CSV file
- Simple command-line interface

📂 File Structure

- "expenses.csv" → Stores all expense data
- Python script → Handles input, storage, and calculations

⚙️ How It Works

- User inputs date, category, amount, and description
- Data is saved in a CSV file
- Program reads data to display and calculate totals

▶️ How to Run

1. Install Python
2. Save the script as "expense_tracker.py"
3. Run using:
   python expense_tracker.py

📖 Learning Outcomes

- File handling in Python
- Working with CSV module
- Basic user input/output
- Simple data processing
